Для запуска приложения запустить файл Triangle.exe
"Результат работы приложения - построение треугольника. Треугольник строится по трем вершинам (А, В, С) . Координаты вершин (X,Y) - целые числа диапазона от -99 до 99. Для каждой стороны треугольника (АВ, ВС, СА) можно выбрать цвет (красный, зеленый, синий) из раскрывающегося списка. При нажатии на кнопку "Построить" открывается диалог "Треугольник" с построенным треугольником по заданным параметрам. 

Задание: 
Функциональное тестирование приложения, написать тестовые случаи. Составить отчет об ошибках.
